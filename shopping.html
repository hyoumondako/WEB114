Questions

  1. Name a situation where you would need to create an "if" statement. (There are many answers to this question. )
  2. When comparing values, in what data type is the result stored? (number, string, or Boolean?)

Answers

  1. When you need to compare values in most any capacity, you can use an "if" statement; for instance, if you need to check if a test score is passing or if you need to validate a user's text input.
  2. Compared values are returned as boolean (true/false.)
